This project is a simple multi-agent system in Python.

It has:
- agents
- an orchestrator
- input handling
- task type formatting
- result handling
- tests for supported tasks
- a test for unknown tasks
- a utils test
- a supported task types test
- a supported check test
- a math error test
- a task-data test
- support for math, text, image, greeting, and echo tasks
- a clear message for unsupported task types
- a README file
